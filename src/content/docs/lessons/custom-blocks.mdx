# Create a Block

Click on the "Blocks" tab.

![Descriptive Alt Text](../lessons/images/lesson-3/blocks-tab.jpg)

Click "New".

![Descriptive Alt Text](../lessons/images/lesson-3/blocks-tab-new-block.jpg)

In the "Sprites" section, click on the grass sprite.

![Descriptive Alt Text](../lessons/images/lesson-3/blocks-tab-grass-sprite.jpg)

Click replace sprite button

![Descriptive Alt Text](../lessons/images/lesson-3/blocks-tab-replace-sprite.jpg)

This will pop up a list of all our sprites. Click the new sprite we made

![Descriptive Alt Text](../lessons/images/lesson-3/blocks-tab-replace-sprite-picker.jpg)

Click "Add Blocks" or "Add Slopes" modes. You click should now be able to see block
under the mouse cursor is using our new block with the custom sprite.

![Descriptive Alt Text](../lessons/images/lesson-3/blocks-new-cursor.jpg)
