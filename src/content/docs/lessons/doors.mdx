# Doors

In Lil Bitty Things, doors are something you make from scratch using the tools given to you.


![Descriptive Alt Text](../lessons/images/lesson-10/teleport-final-demo.gif)


### Create door sprite

Go to the "Sprites" tab and create a new sprite. Click the trash can to clear all pixels.

![Descriptive Alt Text](../lessons/images/lesson-10/new-sprite-clear.jpg)

Draw and name your glorious door.

![Descriptive Alt Text](../lessons/images/lesson-10/draw-door.jpg)

### Create a new Thing

Our object is going to need some interactivity so create a new Thing, swap the sprite and give it a name.

![Descriptive Alt Text](../lessons/images/lesson-10/create-door-thing.jpg)

### Create Teleport Action Sequence

Create a new action sequence and name it "Teleport to Start".

![Descriptive Alt Text](../lessons/images/lesson-10/create-action.jpg)

Add "Teleport" action to our sequence

![Descriptive Alt Text](../lessons/images/lesson-10/add-teleport-action.jpg)

The "Rooms" drop down button lets us select which Room we want to teleport to.

![Descriptive Alt Text](../lessons/images/lesson-10/rooms-drop-down.jpg)

Make sure the "Start" Room is selected.

![Descriptive Alt Text](../lessons/images/lesson-10/rooms-drop-down-start.jpg)

The "Teleport player to", as the name suggests, also lets us control where exactly the player teleports to but with more control.

![Descriptive Alt Text](../lessons/images/lesson-10/rooms-drop-down-start.jpg)

Go to "Rooms" tab and click on "Start" Room.

![Descriptive Alt Text](../lessons/images/lesson-10/change-to-start-room.jpg)

Click on the "Action Sequences" tab. Click "Teleport to Start". Click "Set Player Position"

![Descriptive Alt Text](../lessons/images/lesson-10/click-teleport-picker-start.jpg)

Choose anywhere in the room you want the player to teloport to.

![Descriptive Alt Text](../lessons/images/lesson-10/click-teleport-picker-start-2.jpg)

Go back to "Room 1" and place our door anywhere.

![Descriptive Alt Text](../lessons/images/lesson-10/place-room-1-door.jpg)

Add "Teleport to Start" action to our door.

![Descriptive Alt Text](../lessons/images/lesson-10/add-teleport-to-start-action.jpg)

Now we should be able to teleport to the "Start" Room.

![Descriptive Alt Text](../lessons/images/lesson-10/teleport-to-start-demo.gif)

### Creating return door

We should probably create a return door.

Create a new action in "Action Sequences" tab and name it "Teleport to Room 1".

![Descriptive Alt Text](../lessons/images/lesson-10/create-return-teleport-action.jpg)

Add a "Teleport" action.

![Descriptive Alt Text](../lessons/images/lesson-10/create-return-teleport-action-2.jpg)

In the "Rooms" tab, swtich to "Room 1". Click on "Teleport to Room 1" action in the "Action Sequences" tab.
Click on the "Set Player Position" button and pick somewhere in "Room 1".

![Descriptive Alt Text](../lessons/images/lesson-10/place-return-position.jpg)

In the "Things" tab we have 1 door that teleports us to "Start" Room.

![Descriptive Alt Text](../lessons/images/lesson-10/rename-doors-1.jpg)

Rename our first door "door to start".

![Descriptive Alt Text](../lessons/images/lesson-10/rename-doors-2.jpg)

Duplicate "door to start"

![Descriptive Alt Text](../lessons/images/lesson-10/duplicate-door.jpg)

Rename the duplicate door "door to room 1"

![Descriptive Alt Text](../lessons/images/lesson-10/rename-duplicate-door.jpg)

Select "Teleport to Room 1" action sequence.

![Descriptive Alt Text](../lessons/images/lesson-10/select-teleport-to-room-1-action.jpg)

In the "Rooms" tab, click on "Start" Room.

![Descriptive Alt Text](../lessons/images/lesson-10/go-to-room-1.jpg)

Click on "Place Things" mode on the left tool bar. Then go to the "Things" tab. Click on "teleport to room 1" door and place it
where you want it to go. 

![Descriptive Alt Text](../lessons/images/lesson-10/place-second-door.jpg)

And here's the final result.

![Descriptive Alt Text](../lessons/images/lesson-10/teleport-final-demo.gif)
